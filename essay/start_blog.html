StepUp:基于Pelican的静态博客终于搭建成功了

StepUp：寓意着每天提高一点点。

以前的我，总是不喜笔记，很多的东西过后就忘，

年过40，就开始更健忘了，今天忘东，明天忘西，

渐渐的也就脱离不了笔记了～

可是～～

手绘的笔记不好带呀

本地的电子日记有局限，电脑没有天天带身边

建个Blog要收费，要数据库，要空间，要……

我只是想找个可以自由记录，免费保存

  • 一些想法
  • 一些网摘
  • 一些记录

不需要太多的

  • 评论
  • 交互
  • 反馈
  • 时限
  • 实时

所以 github ，所以 Static Blog

就够了～～

我用 Python ，所以我使用 Pelican来搭建静态博客

几经安装测试，搭建、安装主题、插件，才终于成功，

前面几天一直是基于Python3的虚拟环境在搭建，诸多问题，一直不满意

后来才想到使用基于Python2的虚拟环境来测试，居然一下就成功了，

然后就是

  • 测试主题、插件
  • 测试 pelicanconf.py 的配置
  • 测试各种 make 命令
  • 测试 github pages 的 repository 的申请
  • 测试ssh密钥
  • 测试git推送生成的静态网站

终于，基于Pelican的静态博客终于搭建成功了

没错这就是你看到的网站 StepUp

抢建过程可以参考 用Pelican创建静态博客网站
