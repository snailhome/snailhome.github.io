调教openSUSE Leap 42.1

这两天又抽风把 Linux Mint 系统改装成了 openSUSE Leap 42.1，就着安装使用的情况，做一些记录。

Gnome3 和 KDE5

按默认桌面安装了KDE5，KDE看似自带了很多功能，各个方面都可以配置，但个人感觉带是太花俏，于是有重装成 Gnome3，感觉还是 Gnome3 干净，简洁。

感觉刚装的 openSUSE 比其他发行版带的东西更少，好吧，重新再装起来。

系统更新

openSUSE 安装使用 zypper 命令

好奇怪，刚装的系统更新出错，非得重启以后才能更新

  • 更改国内源并更新软件 我用的是中科大源，直接终端运行以下命令。
sudo zypper mr -da
sudo zypper ar -fc https://mirrors.ustc.edu.cn/opensuse/distribution/leap/42.1/repo/oss USTC:42.1:OSS
sudo zypper ar -fc https://mirrors.ustc.edu.cn/opensuse/distribution/leap/42.1/repo/non-oss USTC:42.1:NON-OSS
sudo zypper ar -fc https://mirrors.ustc.edu.cn/opensuse/update/leap/42.1/oss USTC:42.1:UPDATE-OSS
sudo zypper ar -fc https://mirrors.ustc.edu.cn/opensuse/update/leap/42.1/non-oss USTC:42.1:UPDATE-NON-OSS
sudo zypper ref
sudo zypper up
  • 系统更新
sudo zypper up
  • 搜索软件包，如virutalbox
sudo zypper se virtualbox
  • 安装软件，如Virtualbox
sudo zypper in virtualbox virtualbox-qt

个人必备的gnome扩展

工具 -> 优化工具 -> 扩展 -> 获取得多扩展

安装后可能需要自己设置一下

  • hide top bar —— 全屏自动隐藏顶栏,缺少这个gnome3没法用
  • Clipboard Indicator —— 剪切板
  • Media player indicator —— 显示音乐播放器的状态
  • Battery status —— 显示电池电量的百分比
  • Netspeed —— 在顶栏上显示网速
  • Removable Drive Menu —— 在顶栏显示移动盘图标
  • taskbar —— 顶栏颜色等
  • user themes —— 启用自定义的shell主题
  • Lunar Calendar —— 阴历（依赖opensuse:typelib-1_0-LunarDate-2_0）
  • Todo.txt —— 一个任务/待办项的小工具 你们可以自己找扩展……

安装图标主题

我这里安装的是一个图标主题 numix-icon-theme-circle，opensuse官网下载dvd那里（https://software.opensuse.org/421/zh_CN）搜就行，这样最方便了，（推荐这样，更方便。）

安装外观主题

同图标主题一样，这里搜索一个 arc-theme，然后在 工具 -> 优化工具-> 外观 的 GTK+ 和 Shell主题可以设置这个主题

  • 增加Packman源，安装chromium和pepperflash 像flash、ffmpeg这些非自由版本软件只能通过Packman源安装，flash是上网必备插件这点不用我说吧。
sudo zypper ar http://mirrors.hust.edu.cn/packman/suse/openSUSE_Leap_42.1/ Packman
sudo zypper ref
sudo zypper in chromium chromium-pepper-flash
  • 安装VLC播放器 增加VLC源，用zypper或yast安装
sudo zypper ar http://download.videolan.org/pub/vlc/SuSE/Leap_42.1/ VLC
sudo zypper ref
sudo zypper in vlc
  • 添加输入法 系统自带的五笔输入法是五笔大字集，但候选栏里带常出现方块字，而且不能五笔拼音混打，安装个五笔拼音输入法方便。
sudo zypper in fcitx-table-cn-wubi-pinyin

然后进fcitx的‘配置当前输入法`添加、调整输入法项吧。

安装 WPS

下载RPM文件：http://kdl.cc.ksosoft.com/wps-community/download/a20/wps-office-10.1.0.5444-1.a20.x86_64.rpm 双击下载的RPM文件即可开始安装 首次打开时会提示缺少字体文件，从我的百度云盘下载WPS字体吧 解压后把里面的*.ttf复制到~/.fonts里就可以了。

安装Wine

直接zypper吧

sudo zypper in wine
