Fedora22下安装五笔

Fedora22输入法以ibus为框架，因此要安装五笔输入法，也应该安装安装ibus框架下的极点或海峰五笔。

输入法安装

$ su
Password:
# dnf install -y ibus*wubi*
# reboot

后面一个操作就是重启系统 重启登录系统后：

  • 点击屏幕右上角，点击工具图标，
  • 在打开的窗口里点击个人，点击区域和语言，
  • 点击区域和语言对话框的输入源下的+，
  • 选择添加输入源里的汉语（中国），
  • 选择汉语（极点五笔86（极爽词库 6.0））或汉语（海峰五笔86），点击添加

输入法切换

ibus输入法切换使用WIN键加空格键来切换。
